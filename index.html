About TrueKanban

TrueKanban combines the simplicity of visual work management with powerful flow optimisation techniques. Below are some of the key features that support real Kanban delivery:

  • Work Item Age Tracking: Visual, real-time age of in-progress items
  • Service Level Expectation (SLE) Visualisation: Show SLE boundaries directly on the board and in charts
  • Pull-Based WIP Control: Enforced WIP limits with pull signals when capacity is available
  • Flow Metrics Dashboard: Display the 4 core metrics: WIP, Cycle Time, Throughput, Work Item Age
  • Customisable Definition of Workflow (DoW): Define start/finish points, stages, policies, and flow rules
  • Blocked Item Visualisation: Visibly mark blocked items with cause and duration
  • Forecasting Using Historical Data: Generate probabilistic forecasts based on past cycle times
  • Highlight Ageing Risks: Automatically flag items nearing or exceeding their SLE
  • Work Item Breakdown Support: Split large items into smaller, traceable deliverables
  • Pull System Signals: Visual cues show when teams or workflow stages are ready for more work
